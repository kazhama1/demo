## 📌 Abstract
This project focuses on performing **data cleaning, sorting, and prediction using Microsoft Excel** to analyze raw sales data.

Real-world datasets are often:
- Incomplete  
- Inconsistent  
- Unorganized  

### Key Steps:
- Data Cleaning (remove duplicates, fix formats)
- Sorting & Filtering
- Data Analysis (Pivot Tables, Charts)
- Forecasting (Trendlines & Regression)

### Outcome:
- Improved data accuracy  
- Better insights & decision-making  
- Sales forecasting using Excel tools

## 📊 Results Analysis and Validation

### Step 1: Data Cleaning
- Remove duplicates  
- Handle missing values  
- Fix text formatting (`UPPER()`, `PROPER()`)  
- Format dates  
- Remove extra spaces (`TRIM()`)  

---

### Step 2: Sorting & Filtering
- Sort by:
  - Sales  
  - Region  
  - Date  
- Apply filters:
  - Sales > 1000  
  - Region-specific  

---

### Step 3: Data Analysis
- Pivot Tables:
  - Sales by region  
  - Profit by product  

- Charts:
  - Bar chart → Sales per region  
  - Line chart → Sales over time  

---

### Step 4: Prediction
- Trendline (Linear Regression)  
- Forecast future sales
